# Projekt: Tarakan VR 🪳

[Read in English](README.md) 

## 📝 Koncepcja Gry
Gracz wciela się w rolę karalucha w ogromnym mieszkaniu. Głównym celem jest przetrwanie, pokonywanie przeszkód i ucieczka przed gigantycznym łowcą (Sztuczną Inteligencją), który próbuje zabić gracza za pomocą łapki na muchy lub sprayu owadobójczego.

## 🎯 Kluczowe Mechaniki
* **VR Ruch:** Szybkie poruszanie się z perspektywy karalucha (niska kamera, skala miniaturowa).
* **Łowca AI:** Wróg, który stale śledzi gracza, korzystając z siatki nawigacyjnej (NavMesh).
* **System Przetrwania:** Wykorzystywanie mebli, szczelin i ciasnych miejsc jako kryjówek, do których człowiek nie może dotrzeć.
* **Zagrożenia Środowiskowe:** Lepkie plamy (spowolnienie), okruchy jedzenia (bonusy do prędkości) i przeszkody domowe.

## 🛠 Technologie
* **Silnik:** Unreal Engine 5.4+
* **Platforma:** VR (Meta Quest / SteamVR)
* **Kontrola wersji:** Git + Git LFS
