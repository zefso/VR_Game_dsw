# Project: Tarakan VR 🪳

[Przeczytaj po polsku](README_PL.md)
## 📝 Game Concept
The player takes on the role of a cockroach in a massive apartment. The main goal is to survive, overcome obstacles, and escape from a giant AI-controlled human hunter who tries to kill the player with a fly swatter or insecticide.

## 🎯 Key Mechanics
* **VR Movement:** Fast-paced locomotion from a cockroach's perspective (low camera, miniature scale).
* **AI Hunter:** An enemy that constantly pursues the player using the navigation mesh (NavMesh).
* **Survival System:** Utilizing furniture, cracks, and tight spaces as hiding spots where the human cannot reach.
* **Environmental Hazards:** Sticky puddles (slow down), food crumbs (speed boosts), and household obstacles.

## 🛠 Technical Stack
* **Engine:** Unreal Engine 5.4+
* **Platform:** VR (Meta Quest / SteamVR)
* **Version Control:** Git + Git LFS
